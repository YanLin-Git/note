- 本地搭建流程:
    1. 安装python3
    2. clone本仓库 `git clone https://github.com/yanlinnn/note.git`
    3. 进入文件目录，执行`sh start_docsify.sh`
    4. 浏览器中输入`http://localhost:3000/#/`
